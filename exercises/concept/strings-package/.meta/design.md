# Design

## Goal

The goal of this exercise is to teach the student about the strings package.

## Learning objectives

- Know about the `strings` package (like `Split`, `Index`, `TrimSpace`).

## Out of scope

- iteration of strings
- immutability of strings
- bytes and runes

## Concepts

- `strings`

## Prerequisites

- `slices`
