# Installation

Follow the instructions for your system on the installation page at [golang.org](http://golang.org/doc/install).

## Go version

This track uses Go 1.26.

An earlier version of Go could be used by modifying the per-exercise `go.mod` file and changing the `go 1.26` line.
However, the tests use features introduced in Go 1.22 and will not work without modification with an earlier version.
